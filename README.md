Airbnb NYC 2019 Đ Deep Dive Analysis & Interactive DashboardProject OverviewThis project focuses on performing a deep-dive analysis of the Airbnb NYC 2019 dataset and building an interactive dashboard to uncover business insights. The analysis explores pricing patterns, availability trends, review activity, and borough-level performance using Excel Pivot Tables, Charts, KPIs, and Slicers.Objectives* Define key business KPIs. * Perform segmentation-based analysis. * Explore relationships between important variables. * Build an interactive dashboard for decision-making. * Present actionable business insights. DatasetDataset: Airbnb NYC 2019Records: 48,895 Airbnb listings in New York City.Key attributes:* Borough * Room Type * Price * Availability (365 days) * Number of Reviews * Host Information KPI DefinitionsTotal ListingsTotal number of Airbnb listings available in the dataset.Average PriceAverage listing price across all properties.
Top BoroughBorough with the highest number of listings.Most Common Room TypeMost frequently listed room type.Average AvailabilityAverage number of available days per year.Deep Dive Analysis1. Price Category AnalysisListings were segmented into different price categories to understand market distribution.2. Availability Category AnalysisProperties were categorized based on annual availability to identify listing behavior patterns.3. Review Activity AnalysisReview activity categories were analyzed to understand customer engagement levels.Interactive Dashboard FeaturesThe dashboard includes:* KPI Cards * Listings by Borough * Average Price by Room Type * Room Type Distribution * Availability by Borough * Price Category Distribution * Borough Slicer * Room Type Slicer The dashboard allows users to interactively filter and explore data.Key Insights* Manhattan has the highest number of Airbnb listings. * Entire Home/Apt is the most common room type. * Entire Home/Apt listings have the highest average price. * Listing availability varies significantly across boroughs. * Most properties fall within the medium price category. * Price and availability show only a weak relationship. * Segmentation analysis helps identify market patterns and customer behavior. Tools Used* Microsoft Excel * Pivot Tables * Pivot Charts * Slicers * Scatter Plot Analysis * Dashboard DesignAUTHORAjay Nath
